
اسم المشروع: جمعية تسيير بئر الخلالقة

الوصف:
هذا المشروع يحتوي على ملفات الموقع الرسمي لجمعية تسيير بئر الخلالقة. يشمل صفحات للمنخرطين، الأداءات، الاشتراكات، الصيانة، إدارة الجمعية، معلومات الاتصال، وأخبار الجمعية. الهدف من المشروع هو تسهيل إدارة الجمعية إلكترونيًا وتوفير تجربة واضحة وسهلة لجميع المستخدمين.

المميزات:

إدارة المنخرطين وتتبع الاشتراكات والأداءات.

لوحة إدارة موحدة لطلبات الانخراط والأداءات.

صفحات معلوماتية وروابط سريعة للمستخدمين.

بانر متحرك وصور/إيموجي متحركة.

تصميم متجاوب يصلح لجميع الشاشات.
